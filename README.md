## macOS dotfiles

### How to use
* First create your ssh key "ssh-keygen -t rsa -b 4096 -C "your_email@example.com""
* Clone this repo (use ssh uri, you can use recently created public key in your account for the computer)
* To use flutter, download flutter SDK and put it in your home directory, or remove .bash_extra_flutter
* Instal homebrew
* Copy contents to your home directory
* In home directory hit "brew bundle"

## What's included?
* My traditional bash_profile, bashrc and bash_prompt with bash_extra_psql
* Includes: nvm, git, pyenv, composer, postgresql, google-chrome, avast-security, keepingyouawake, macs-fan-control, android-sdk, flutter, etc..
* Include IDEs: Android Studio, PhpStorm, PyCharm, WebStorm
* Also Includes: Sublime, Lightshot Screenshot, Slack
